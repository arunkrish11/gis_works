# GIS Projects using QGIS

## Overview
This repository contains a collection of GIS exercises performed using QGIS, focusing on satellite image processing, spatial analysis, and map creation.

These projects demonstrate practical skills in handling raster and vector data, creating thematic maps, and extracting geographic features.

---

## Tools & Data Sources
- QGIS  
- Google Earth  
- OpenStreetMap  
- USGS Earth Explorer  

---

## Projects

---

### 1. Creating Colour Composites

**Objective:**  
Generate True Color and False Color composites using Landsat satellite imagery.

**Key Work:**
- Downloaded Landsat 8/9 data from USGS  
- Combined raster bands using Virtual Raster  
- Created:
  - True Color Composite (natural appearance)  
  - False Color Composite (vegetation highlighted)

**Output:**  
- True color image  
- False color image  

---

### 2. Combining Two Tiles and Creating Colour Composites

**Objective:**  
Process multi-tile satellite data and analyze temporal changes in Malappuram district.

**Key Work:**
- Downloaded Landsat data (before 2000 & 2020)  
- Created virtual rasters for each tile  
- Merged tiles (mosaicking)  
- Clipped to study area using shapefile  
- Generated False Color composites  

**Output:**  
- Pre-2000 and 2020 comparison maps  

---

### 3. Extraction of Water Bodies and Rivers

**Objective:**  
Digitize and extract water bodies and rivers from satellite imagery.

**Key Work:**
- Created False Color Composite  
- Digitized:
  - Water bodies (polygon shapefile)  
  - Rivers (line shapefile)  
- Applied symbology and exported shapefiles  

**Output:**  
- Extracted water body and river layers  

---

### 4. Map Creation and Layout Design using QGIS

**Objective:**  
Create a professional map layout with cartographic elements.

**Key Work:**
- Imported shapefile  
- Designed map layout using Print Layout  
- Added:
  - Coordinates grid  
  - Legend  
  - Scale bar  
  - North arrow  

**Output:**  
- Final map image ready for presentation  

---

### 5. Georeferencing a Shapefile in QGIS

**Objective:**  
Align spatial data with real-world coordinates.

**Key Work:**
- Added Ground Control Points (GCPs)  
- Georeferenced shapefile using reference map  
- Adjusted transparency for overlay  

**Output:**  
- Georeferenced spatial data  

---

### 6. Creating a Campus Map using Google Earth and QGIS

**Objective:**  
Digitize real-world features and create a complete map layout.

**Key Work:**
- Marked features (trees, roads, buildings) in Google Earth  
- Exported data as KML  
- Imported into QGIS  
- Created map layout with:
  - Legend  
  - Scale bar  
  - North arrow  

**Output:**  
- Campus map with labeled features  

---

## Key Skills Demonstrated

- Satellite image analysis  
- Mosaicking and clipping  
- Shapefile creation and digitization  
- Map layout and cartographic design  
- Georeferencing techniques
